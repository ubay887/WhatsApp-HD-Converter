🔧 TROUBLESHOOTING

Server tidak start:
  • Cek port 5000 tidak digunakan aplikasi lain
  • Restart terminal/command prompt
  • Jalankan sebagai Administrator

UI tidak muncul:
  • Refresh browser (Ctrl+F5)
  • Cek console browser (F12) untuk error
  • Pastikan folder templates/ dan static/ ada

Upload error:
  • Cek ukuran file (max 100MB)
  • Cek format file yang didukung
  • Lihat terminal untuk error message

FFmpeg error:
  • Restart terminal/command prompt
  • Cek: ffmpeg -version
  • Restart komputer jika perlu
